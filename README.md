# ProRaschet
Бот-ассистент для бухгалтера. Помогает быстро находить ответы на бухгалтерские вопросы со ссылками на законодательство

- **Инструменты и технологии, использованные в проекте**: Telegram  API, BotFather, n8n, OpenAI Chat Model, Timeweb
- **Ассистент предоставляет структурированные строгие ответы** и указания на нормы законодательства в ответе
